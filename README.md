🚀 100 Days – 100 Python Projects Challenge
===========================================

Welcome to my **#100Days100Projects Challenge**!In this journey, I will build **100 Python projects in 100 days** to strengthen my programming fundamentals and improve problem-solving skills.

This challenge is focused on **learning by building**, starting from beginner concepts and gradually moving toward more practical and advanced projects.

The goal is to:

*   Improve Python programming skills 🐍
    
*   Build consistent daily coding habits
    
*   Create a portfolio of real projects
    
*   Practice core programming concepts step by step
    

🎯 Challenge Goals
==================

*   📅 Build **1 project every day for 100 days**
    
*   🧠 Practice **core Python concepts**
    
*   💡 Learn by solving **real-world problems**
    
*   📂 Maintain a structured **GitHub project portfolio**
    

📚 Topics Covered
=================

During this challenge, I will work on projects covering:

*   Variables & Data Types
    
*   User Input & Output
    
*   Conditional Statements (if-else)
    
*   Loops (for & while)
    
*   Functions
    
*   Lists, Dictionaries & Sets
    
*   File Handling
    
*   Object-Oriented Programming
    
*   APIs & Automation
    
*   CLI Tools & Mini Applications
    

🛠️ Tech Stack
